DEFINITION - SUB
    SaveBin file$, addr&, size& -- Save GP memory contents into a file

OPERATION
    SaveBin is a simple memory saving routine.  It can deal with arbitrary
    lengths and its pointer on arbitrary alignments.  It simply writes the
    memory contents as is into the given file.

ARGUMENTS
    file$ -- the name of the file, if required inlusive a full path
    addr& -- a address pointer to the memory block to write out
    size& -- the size (in bytes) of the memory block
