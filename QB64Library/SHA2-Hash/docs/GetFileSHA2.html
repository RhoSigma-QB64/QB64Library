GetFileSHA2$ (function)

Compute the SHA2 Message-Digest of the specified file. The file can contain any text or binary data.

SYNTAX:

digest$ = GetFileSHA2$ (FileSpec$)

INPUTS:

FileSpec$ (STRING)

  • The path/file specification of the file you want to get the SHA2 Message-Digest from.

RESULT:

digest$ (STRING)

  • The SHA2 Message-Digest of the given file as hexadecimal string.
  • Will be an empty string, if the file could not be opened (either not found or not accessible).
