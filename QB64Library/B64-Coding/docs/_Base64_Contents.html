Base64 En-/Decoding

Perform Base64 encoding and decoding of any arbitrary string message data. There are options to encode to Base64 in multiple ways. Decoding can either enforce valid Base64 strings or it can be tolerant on invalid chars. This library is based on the sources used in the MIME plugin of Notepad++, which is under the terms of the GNU GPL v2, see subfolder B64-Coding\license.
